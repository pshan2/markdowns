##Attribute Needed

Start from `<channel>/<item>`:
	
###Title
`<title>`: title of feed

###Link

Three Attributes: `<orgiLink>`/`<link>`/`<guid isPermaLink>="false"`: One of them needs to be the real link 
	
1. `<orgiLink>` does not exist
	
2. `<link>` starting from http://feedproxy.google.com

3. : if two cases both happens, use `<guid isPermaLink>`
	
###Date

`<pubDate>`: sometimes use this value to decide feed sequence -> Transfer to integer

###Description

`<description>`: proxy.cfm can deliver all content. In JavaScript end, need to parse:
	
1. if `<![CDATA[` appears, removing it and end tag

2. Escape code such as `&amp;#8211;`. Put a JavaScript escaping library in CPA-Framework for future usage:https://cascade.emory.edu/entity/open.act?id=fe581d970ae7232f0873db242354c54d&type=file

3.
